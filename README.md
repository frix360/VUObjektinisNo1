## Įdiegimas (Unix kompiuteryje) naudojant `Clang`kompiliatorių 

- `git clone https://github.com/frix360/VUObjektinisNo1.git`
- `cd VUObjektinisNo1`
- `clang++ -std=c++11 main.cpp -o main`
- `./VUObjektinisNo1`
